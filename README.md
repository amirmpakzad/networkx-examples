# NetworkX Examples

### 🔬 Project Overview

This project demonstrates:
- Data preprocessing pipelines for network analysis
- Visualization of graph embeddings using Matplotlib and SciPy
- Converting **NetworkX** graphs into **PyTorch** tensors
- Seamless **NetworkX & PyTorch** integration for Graph Representation Learning

---

## 🛠 Prerequisites

- **Python**: 3.9 or higher

---

## 🚀 Getting Started

### 1. Environment Setup

It is highly recommended to use a virtual environment to manage dependencies and avoid version conflicts.

```bash
# Create a virtual environment
python -m venv venv

# Activate the environment

# Windows:
venv\Scripts\activate

# Linux / macOS:
source venv/bin/activate

```



### 2. Dependency Installation

```bash
# Upgrade pip
pip install --upgrade pip

# Install required libraries
pip install torch matplotlib scipy networkx
```

In case of connectivity issues with PyPI, you can use the Runflare mirror:
```bash
# Install required libraries
pip install -i https://mirror-pypi.runflare.com/simple torch matplotlib scipy networkx
```
